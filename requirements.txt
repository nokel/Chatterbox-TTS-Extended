# Chatterbox-TTS-Extended - single requirements file for the whole project
# (web app, headless TTS API server, and the Discord voice bot).
#
# GPU backend (torch/torchaudio) is installed separately, BEFORE this file,
# by install.ps1 based on the detected hardware (ROCm wheels for AMD, CUDA
# wheels for NVIDIA, CPU wheels otherwise). torch must NOT be listed here or
# pip would replace the hardware build with the generic CPU one.
#
# NOTE: faster-whisper pulls in the NVIDIA-only ctranslate2 from PyPI; on AMD
# install.ps1 replaces it afterwards with OpenNMT's official ROCm Windows
# wheel so faster-whisper runs on the AMD GPU too.

#   AMD     onnxruntime-webgpu   (DirectX 12; implements GroupQueryAttention,
#                                 which the DirectML build cannot run)

#   Intel   onnxruntime-openvino + openvino==2025.4.*  (versions must pair:
#                                 1.24.x <-> 2025.4.*; a mismatch loads but
#                                 silently falls back to CPU)
#   CPU     onnxruntime
